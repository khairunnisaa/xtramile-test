# Patient MPI Frontend

Angular frontend application for Patient Master Patient Index system.

## Tech Stack
- Angular 17
- Angular Material
- RxJS
- TypeScript

## Prerequisites
- Node.js 18+
- npm or yarn

## Installation
```bash
npm install
```

## Development
```bash
ng serve
```

Application runs on `http://localhost:4200`

## Build
```bash
ng build
```

## Features
- Patient CRUD operations
- Server-side pagination
- Debounced search
- Material Design UI
